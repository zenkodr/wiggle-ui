# What is a Widget?

A flexible container component used to build UI widgets. It supports multiple sizes, design variations, color variants, and includes a collection of sub-components to structure your layout.
Widgets are built using **class-variance-authority** for consistent styling and composability.

# Widget.tsx

Every widget is directly dependent on the `widget.tsx` file. This file contains the core logic for the widget, including the logic for rendering the widget and handling user interactions. This widget file exposes a few **Variants** and **Components** that you can use to customize the widget's appearance and behavior.

# Sub Components

The following components are structural wrappers and accept standard HTML attributes and className.

- <span className="bg-secondary rounded-md px-2 py-1">WidgetHeader</span> : Top
  section, flex container (justify-between).

- <span className="bg-secondary rounded-md px-2 py-1">WidgetTitle</span> : Styled `<h5>` for the widget heading.

- <span className="bg-secondary rounded-md px-2 py-1">WidgetContent</span> :
  Center section, grows to fill available space (flex-1), centers children.

- <span className="bg-secondary rounded-md px-2 py-1">WidgetFooter</span> :
  Bottom section, flex container.
